Types of Diagrams

  • Dot Plot aka Dot Diagram is suitable for small samples up to about 20 in size.
  • Stem-and-Leaf Diagram is suitble for moderate sized data sets 20≤n≤10020\le n\le 10020≤n≤100.

  • Normal or Gaussian Distribution is the most widely used model for randomness variation in data.

Constructing Stem and Leaf Diagram

  1. Divide each number in the sample to stem and leaf
    1. Stem consists of one or more leading digits
    2. Leaf consists of the one single remaining digit
  2. List stems values in a vertical column
  3. Record all leaf values in its corresponding stem row.

Warnings:

  • Must include legend for stem-and-leaf diagram.
    • Title and unit name
    • Sample size, e.g. n=80n=80n=80
    • Leaf unit = 1
  • If here are too few stems, divide into lower and upper stems.
    • Lower L (with leaves 0-4); Upper U (with leaves 5-9)
    • In extreme cases, use Z(0-1), T(2-3), F(4-5), S(6-7), E(8-9)

Constructing a Box Plot

  • 2 boxes are bounded by q1q_1q1​, q2q_2q2​ and q3q_3q3​.
  • Line extends to two whiskers.
  • Outliers are shown as dots.

Sample Space and Events

Sample Space: all possible outcomes of a test, denoted by set SSS.

Event: a subset of sample space, denoted by set EEE.

Probability of an event is denoted P(E)P(E)P(E), 0≤P(E)≤10\le P(E)\le 10≤P(E)≤1.

Control and Non-controlled Variables

Control Variable are kept the same during a Random Experiment, e.g. room temperature, test procedure.

Non-Controlled Variable are factors impacts the experiment outcome and result in slight differentiations, e.g. uncertainty, error in test equipment, background noises.

Discrete Random Variables's range is a finite, countable set of values.
