Constructing Stem and Leaf Diagram

  1. Divide each number in the sample to stem and leaf
    1. Stem consists of one or more leading digits
    2. Leaf consists of the one single remaining digit
  2. List stems values in a vertical column
  3. Record all leaf values in its corresponding stem row.

Warnings:

  • Must include legend for stem-and-leaf diagram.
    • Title and unit name
    • Sample size, e.g. n=80n=80n=80
    • Leaf unit = 1
  • If here are too few stems, divide into lower and upper stems.
    • Lower L (with leaves 0-4); Upper U (with leaves 5-9)
    • In extreme cases, use Z(0-1), T(2-3), F(4-5), S(6-7), E(8-9)

Constructing a Box Plot

  • 2 boxes are bounded by q1q_1q1​, q2q_2q2​ and q3q_3q3​.
  • Line extends to two whiskers.
  • Outliers are shown as dots.

Poisson Distribution is the distribution for discrete data over an interval/range of values.

Poisson Process

A random experiment with the following properties is called a Poisson process:

Consider an interval TTT of real numbers partitioned into subintervals of smaller length Δt\Delta tΔt and assume that as Δt\Delta tΔt tends to zero.

  1. the probability of more than one event in Δt\Delta tΔt tends to zero
  2. the probability of one event in a subinterval tends to λΔt/T\lambda \Delta t/TλΔt/T
  3. the event in each subinterval is independent of other subintervals.

These assumptions imply that the subintervals can be thought of as approximately independent Bernoulli trials with success probability p=λΔt/Tp=\lambda \Delta t/Tp=λΔt/T and the number of trials equal to n=T/Δtn=T/\Delta tn=T/Δt. Here, the expected value λ=pn\lambda = pnλ=pn, and as Δt\Delta tΔt tends to zero, nnn tends to infinity.

Experimental Uncertainty

Uncertainty analysis is performed during design phase of experiment to estimate uncertainty in final results due to systematic cause and random variations. The goal is to plan experiment.

The validity of measurement is a measure of how closely the output of a measurement system matches the actual measured value. An Error is defined as Measured Value minus True Value. However, the true value is usually not known. Therefore, we replace error with uncertainty.

Uncertainty: estimate with some level of confidence of the limits of error in a measurement.

Systematic Error: fixed, bias error, consistent, repeatable. Its sources include:

  • Calibration Error: non-linear, hysteresis, etc.
  • Loading Error: intrusive measurement device affects measurement
  • Spatial Error: sensor affected by environment.

Random Error: lack of repeatability, background noise, uncontrolled variables, etc.

  • Hysteresis: different measured values from increasing/decreasing loads, recall hysteresis loop (systematic)
  • Non-repeatability is random error

Random Errors Calibration

Calibration: comparison of measured value with a standard, or another measurement system of known accuracy.

Calibration procedure:

  1. Take a set of measurements using measurement system
  2. Obtain Calibration Curve where true value is yyy axis, and measured value is x axis. Fit data with line, curve. Equation of best fit line/curve is calibration equation.
  3. Use calibration equation to calculate measured values
  4. Calculate accuracy, maximum uncertainty using deviation plot. Deviation=true value−measured valueDeviation = true\ value - measured\ valueDeviation=true value−measured value. In deviation plot, deviation is y axis, true value is x axis.

In short,

  • Calibration Curve: true value vs measured value
  • Deviation Plot: deviation vs true value
  • Max Uncertainties: lowest and highest deviation

Systematic error: look for patterns in deviation data
Random error: lack of pattern, scattered for repeat tests
