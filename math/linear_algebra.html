Subspaces

A non-empty subset SSS of Rn\mathbb{R}^nRn is called a subspace of Rn\mathbb{R}^nRn if

∀x⃗,y⃗∈S∧t∈R,x⃗+y⃗∈S∧tx⃗∈S\forall \vec{x},\vec{y}\in S \wedge t\in \mathbb{R}, \vec{x}+\vec{y}\in S\wedge t\vec{x}\in S ∀x,y​∈S∧t∈R,x+y​∈S∧tx∈S

In other words, all vectors in the subspace SSS are closed under addition and closed under scalar multiplication.

{0⃗}\{\vec{0}\}{0} in Rn\mathbb{R}^nRn is called a trivial subspace.

Systems of Linear Equations and Matrix

Coefficient Matrix and Augmented Matrix

Coefficient Matrix, denoted by AAA, is a matrix representing the coefficients of variables in a system of linear equations. This does not include the constant term on the right hand side.

Augmented Matrix, denoted by [ A ∣ b ⃗][\,A\,|\,\vec{{b}\,}][A∣b], is a matrix that represents the complete system of linear equations, which includes the constant term on the right hand side (represented by b⃗\vec{b}b).

Note that when we talk about m×nm\times nm×n matrix, we are referring to coefficient matrix (excluding the b⃗\vec{b}b term). This means that mmm is the number of linear equations. nnn is the number of variables.

In other words, a m×nm\times nm×n matrix refers to a linear system of mmm equations in nnn unknowns.

Row Echelon Form

A leading entry is the first non-zero entry in a matrix.

A matrix is in row echelon form (REF) if:

  1. All-zero rows are at the bottom
  2. Comparing two non-zero rows, the upper row's leading entry is to the left of the lower row's leading entry.

If a system of equations is obtained from another system by one or more elimination steps, the systems are said to be equivalent. If the matrix MMM is reduced into matrix NNN by a series of elementary row operations. then we say MMM is row equivalent to NNN.

A matrix is said to be in reduced row echelon form (RREF) if:

  1. It is in row echelon form.
  2. All leading entries are 1, called a leading 1.
  3. In a column with a leading 1, all the other entries are 0s.

For any given matrix, there is a unique matrix in reduced row echelon for that is equivalent to AAA.

Rank and Consistent Systems

The rank of a matrix is the number of leading 1s in its RREF.

A system that has at least one solution is called consistent. A system that has no solution is called inconsistent.

To visualize this, if the hyperplanes (rows) do not have a common intersection, the system is inconsistent. Otherwise, the system is consistent.

A system is inconsistent if and only if some row of [ S  ∣  c⃗ ][\,S\;|\;\vec{c}\,][S∣c] is of the form [ 0  0  ...  0  ∣  c ][\,0\;0\;...\;0\;|\;c\,][00...0∣c].

A system is consistent if and only if the rank of the coefficient matrix is equal to the rank of the augmented matrix.

If the system is consistent, then

#  parameters=#  unknowns−rank(A)\#\; parameters = \#\;unknowns-rank(A) #parameters=#unknowns−rank(A)

# parameters also represents the dimension number.

Let [ A ∣ b⃗ ][\,A\,|\,\vec{b}\,][A∣b] be a system of mmm linear equations in nnn variables. Then [ A ∣ b⃗ ][\,A\,|\,\vec{b}\,][A∣b] is consistent for all b⃗∈Rn\vec{b}\in\mathbb{R}^nb∈Rn if and only if rank(A)=mrank(A)=mrank(A)=m.

Homogeneous Linear Equations

Homogeneous Linear Equation refers to a linear equation whose right-hand side is zero.

Homogeneous System is a system of linear equations in which all the equations are homogeneous.
