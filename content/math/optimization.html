Introduction

An optimization is a problem of the form

{minimize(f(x):x∈S)maximize(f(x):x∈S)\begin{cases} \textrm{minimize}(f(x):x\in S)\\ \textrm{maximize}(f(x):x\in S) \end{cases} {minimize(f(x):x∈S)maximize(f(x):x∈S)​

  • Feasible Region is set SSS
  • Objective Function: f:S→Rf:S\rightarrow \mathbb{R}f:S→R

Note that min⁡(f(x):x∈S)=max⁡(−f(x):x∈S)\min(f(x):x\in S) =\max(-f(x) : x\in S)min(f(x):x∈S)=max(−f(x):x∈S)

  • Feasibility: we say that min⁡(f(x):x∈S)\min(f(x):x\in S)min(f(x):x∈S) is feasible if SSS is non-empty
  • Boundedness: we say that min⁡(f(x):x∈S)\min(f(x):x\in S)min(f(x):x∈S) is bounded if there exists b∈Rb\in\textrm{R}b∈R such that f(x)≥bf(x)\ge bf(x)≥b for each x∈Sx\in Sx∈S. This definition is analogues to the formal definition of big O notation.
  • Ill-Posedness: min⁡(x:x>1)\min(x:x>1)min(x:x>1) is both feasible and bounded, but there is no optimal solution.

Axiom (L1-1): if a set is non-empty and bounded below, then there exists a greatest lower bound

For minimization problems: if min⁡(f(x):x∈S)\min(f(x):x\in S)min(f(x):x∈S) is both feasible and bounded, then we can find an optimal bound

  • Infimum: inf⁡(f(x):x∈S)=max⁡(b∈R:b≤f(x) ∀ x∈S)\inf(f(x):x\in S)=\max(b\in \mathbb{R}:b\le f(x)\ \forall\ x\in S)inf(f(x):x∈S)=max(b∈R:b≤f(x) ∀ x∈S)

    inf⁡(f(x):x∈S)≔{∞if infeasible−∞if unbounded\displaystyle \inf(f(x):x\in S) \coloneqq \begin{cases} \infty & \textrm{if infeasible}\\-\infty &\textrm{if unbounded}\end{cases} inf(f(x):x∈S):={∞−∞​if infeasibleif unbounded​

    • cone(S)\textrm{cone}(S)cone(S) denotes the smallest cone containing S⊆RnS\subseteq \mathbb{R}^nS⊆Rn. This is well-defined.

Theorem (L2-1): given any collection of convex sets, their intersection is itself a convex set.

Proof: if the intersection is empty, then L2-1 is vacuously true. Otherwise, take any two points A,B in the intersection. The line AB joining these points must also lie wholly within each set in the collection, hence must lie wholly within their intersection.

Claim (L2-2): For any set X⊆RnX\subseteq \mathbb{R}^nX⊆Rn, there is a unique minimal convex set containing XXX.

Proof: take the intersection of all convex sets containing X.
