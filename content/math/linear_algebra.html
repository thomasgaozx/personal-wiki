Cross Product

We only worry about cross product in a 3D space, consider v⃗×w⃗\vec{v}\times \vec{w}v×w

[v1v2v3]×[w1w2w3]=∣i^v1w1j^v2w2k^v3w3∣\begin{bmatrix}v_1\\v_2\\v_3\end{bmatrix} \times \begin{bmatrix}w_1\\w_2\\w_3\end{bmatrix} =\begin{vmatrix} \hat{i} & v_1 & w_1\\ \hat{j} & v_2 & w_2\\ \hat{k} & v_3 & w_3 \end{vmatrix} ⎣⎢⎡​v1​v2​v3​​⎦⎥⎤​×⎣⎢⎡​w1​w2​w3​​⎦⎥⎤​=∣∣∣∣∣∣∣​i^j^​k^​v1​v2​v3​​w1​w2​w3​​∣∣∣∣∣∣∣​

Anyone would ask, why are there vectors (i^,j^,k^\hat{i}, \hat{j}, \hat{k}i^,j^​,k^) in a matrix? A better way to deconstruct this is to picture a function that takes in an arbitrary vector u⃗=(x,y,z)\vec{u} = (x,y,z)u=(x,y,z) as input, and output the volume spanned by u⃗,v⃗,w⃗\vec{u}, \vec{v}, \vec{w}u,v,w:

f([xyz])=∣xv1w1yv2w2zv3w3∣f\left(\begin{bmatrix}x\\y\\z\end{bmatrix}\right) =\begin{vmatrix} x & v_1 & w_1\\ y & v_2 & w_2\\ z & v_3 & w_3 \end{vmatrix} f⎝⎜⎛​⎣⎢⎡​xyz​⎦⎥⎤​⎠⎟⎞​=∣∣∣∣∣∣∣​xyz​v1​v2​v3​​w1​w2​w3​​∣∣∣∣∣∣∣​

We can easily prove that fff is linear, and therefore fff can be represented by a 1×31\times 31×3 matrix mapping [ p⃗⊺ ][\ \vec{p}^\intercal\ ][ p​⊺ ] (which is equivalent to dot product):

[p1p2p3]⋅[xyz]=∣i^v1w1j^v2w2k^v3w3∣\begin{bmatrix}p_1\\p_2\\p_3\end{bmatrix} \cdot \begin{bmatrix}x\\y\\z\end{bmatrix} =\begin{vmatrix} \hat{i} & v_1 & w_1\\ \hat{j} & v_2 & w_2\\ \hat{k} & v_3 & w_3 \end{vmatrix} ⎣⎢⎡​p1​p2​p3​​⎦⎥⎤​⋅⎣⎢⎡​xyz​⎦⎥⎤​=∣∣∣∣∣∣∣​i^j^​k^​v1​v2​v3​​w1​w2​w3​​∣∣∣∣∣∣∣​

Turns out thats

{p1=v2w3−v3w2p2=v3w1−v1w3p3=v1w2−v2w1\begin{cases} p_1=v_2 w_3 - v_3 w_2\\ p_2=v_3 w_1 - v_1 w_3\\ p_3=v_1 w_2 - v_2 w_1 \end{cases} ⎩⎪⎪⎨⎪⎪⎧​p1​=v2​w3​−v3​w2​p2​=v3​w1​−v1​w3​p3​=v1​w2​−v2​w1​​

Turns out p⃗\vec{p}p​ is the vector we are looking for. It is the vector the size of the area spanned by v⃗,w⃗\vec{v}, \vec{w}v,w, with the direction that's normal to the plane.

To summarize the idea here, what we're doing is simply describing cross product's geometric properties as some sort of linear, matrix transformation for an arbitrary vector u⃗\vec{u}u.

  • Coefficient Matrix AAA only contains lhs coefficients, no rhs constants.
  • Augmented Matrix [A∣b⃗][A|\vec{b}][A∣b] has both lhs coefficients (AAA), and rhs constants (b⃗\vec{b}b)

Note that when we say m×nm\times nm×n matrix, we mean mmm rows, nnn columns.

Row Echelon Form

A matrix is in row echelon form (REF) if:

  1. All-zero rows are at the bottom
  2. Comparing two non-zero rows, the upper row's leading entry is to the left of the lower row's leading entry.

A matrix is said to be in reduced row echelon form (RREF) if:

  1. It is in row echelon form.
  2. All leading entries are 1, called a leading 1.
  3. In a column with a leading 1, all the other entries are 0s.

Two m×nm\times nm×n matrices A,BA,BA,B are row equivalent if:

  1. AAA can transform to BBB via elementary row operations.
  2. Row Space equivalence: Row(A)=Row(B)\mathrm{Row}(A)=\mathrm{Row}(B)Row(A)=Row(B)

Rank and Consistency

  • Rank of a matrix is the number of leading 1s in its RREF.
  • Consistent System: a system that has at least one solution, i.e. hyperplanes have common intersect
  • Inconsistent System [ S ∣ c⃗ ][\ S\ |\ \vec{c}\ ][ S ∣ c ] has a row of the form [ 0⃗⊺ ∣ c ][\ \vec{0}^\intercal \ |\ c\ ][ 0⊺ ∣ c ].

There are several properties recarding system consistency:

  • If the system is consistent:

    #dimensions=#parameters=#unknowns−rank(A)\#\mathrm{dimensions}=\#\mathrm{parameters} = \#\mathrm{unknowns}-\mathrm{rank}(A) #dimensions=#parameters=#unknowns−rank(A)

  • The system Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b is consistent if and only if b⃗∈Range(L)\vec{b}\in \mathrm{Range}(L)b∈Range(L) or equivalently b⃗∈Col(A)\vec{b}\in \mathrm{Col}(A)b∈Col(A)

Rank Theorem

  • Vector Space Rn\mathbb{R}^nRn consists of all nnn-dimensional vectors.
  • Row Space of matrix AAA is the span of all row vectors:

    Row(A)={ A⊺x⃗∈Rn ∣x⃗∈Rm }\mathrm{Row}(A)=\{\ A^\intercal\vec{x}\in \mathbb{R}^n\ | \vec{x}\in \mathbb{R}^m\ \} Row(A)={ A⊺x∈Rn ∣x∈Rm }

  • Column Space of matrix AAA is the span of all column vectors:

    Col(A)={ Ax⃗∈Rm ∣x⃗∈Rn }\mathrm{Col}(A)=\{\ A\vec{x} \in \mathbb{R}^m\ | \vec{x}\in \mathbb{R}^n\ \} Col(A)={ Ax∈Rm ∣x∈Rn }

  • Solution Space of m×nm\times nm×n matrix AAA is

    S={ x⃗∈Rn ∣Ax⃗=0⃗ }S=\{\ \vec{x}\in \mathbb{R}^n\ | A\vec{x}=\vec{0}\ \} S={ x∈Rn ∣Ax=0 }

  • Null Space/Kernel of linear mapping LLL is

    Null(L)={ x⃗∈Rn ∣L(x⃗)=0⃗ }\mathrm{Null}(L)=\{\ \vec{x}\in \mathbb{R}^n\ | L(\vec{x})=\vec{0}\ \} Null(L)={ x∈Rn ∣L(x)=0 }

  • Nullity of matrix AAA is the dimension of the nullspace of AAA, denoted by

    nullity(A)\mathrm{nullity}(A) nullity(A)

See that the solution space is literally the set of all solutions of the system, which could be visualized as the intersection of all mmm hyperplanes in the system. The fact that the system is homogeneous means the solution space passes through the origin.

Suppose p⃗,v⃗∈Rn\vec{p}, \vec{v}\in \mathbb{R}^np​,v∈Rn are any 2 solutions to Ax⃗=b⃗A\vec{x} = \vec{b}Ax=b where b⃗≠0⃗\vec{b}\neq\vec{0}b​=0, then p⃗−v⃗\vec{p}-\vec{v}p​−v is a solution to Ax⃗=0⃗A\vec{x} = \vec{0}Ax=0. Similarly, suppose h⃗\vec{h}h is a solution to system Ax⃗=0⃗A\vec{x}=\vec{0}Ax=0, then p⃗+h⃗\vec{p}+\vec{h}p​+h is a solution to system Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b.

Rank Theorem states that for an m×nm\times nm×n matrix AAA:

  1. Dimension of  Row(A)=rank(A)\textrm{Dimension of }\ \mathrm{Row}(A)=\mathrm{rank}(A)Dimension of  Row(A)=rank(A)
  2. Dimension of  Col(A)=rank(A)\textrm{Dimension of }\ \mathrm{Col}(A)=\mathrm{rank}(A)Dimension of  Col(A)=rank(A)
  3. rank(A)+nullity(A)=n\mathrm{rank}(A)+\mathrm{nullity}(A)=nrank(A)+nullity(A)=n

The row space and column space of matrix AAA have the same dimension, equal to rank AAA.

Matrix as Spatial Transform

Consider 2D vector x⃗=x1i^+x2j^\vec{x}=x_1 \hat{i}+x_2 \hat{j}x=x1​i^+x2​j^​, if we were to transform the vector linearly[ref] such that

  • Basis i^\hat{i}i^ is transformed to a new basis L(i^)L(\hat{i})L(i^)
  • Basis j^\hat{j}j^​ is transformed to a new basis L(j^)L(\hat{j})L(j^​)

It helps to picture the linearly stretched, rotated, and contorted space with the new bases. It follows that x⃗\vec{x}x is transformed to vector L(x⃗)L(\vec{x})L(x):

L(x⃗)=x1L(i^)+x2L(j^),L(x⃗)=[ L(i^)L(j^) ]x⃗L(\vec{x})=x_1 L(\hat{i}) + x_2 L(\hat{j}), \qquad L(\vec{x})=[\ L(\hat{i})\quad L(\hat{j})\ ]\vec{x} L(x)=x1​L(i^)+x2​L(j^​),L(x)=[ L(i^)L(j^​) ]x

This demonstrates the nature of an n×nn\times nn×n matrix AAA; it is a function (aka mapping) that:

  1. Takes a n-dimensional vector as input.
  2. Transform the function linearly.
  3. Split out the transformed vector.

The iiith column of a matrix is what the iiith standard basis e^i\hat{e}_ie^i​ becomes after the transformation: L(e^i)L(\hat{e}_i)L(e^i​). When we know what all of the standard basis will transform into, we have sufficient information to know what vector x⃗\vec{x}x will become after the transformation. Namely,

L(x⃗)=∑i=1nxiL(e^i)=[ L(e^1)⋯L(e^n) ]x⃗L(\vec{x})=\sum_{i=1}^n x_i L(\hat{e}_i)=[\ L(\hat{e}_1)\quad\cdots\quad L(\hat{e}_n)\ ]\vec{x} L(x)=i=1∑n​xi​L(e^i​)=[ L(e^1​)⋯L(e^n​) ]x

So now, when you see Ax⃗=v⃗A\vec{x}=\vec{v}Ax=v, you should know by reflex, that the n×nn\times nn×n matrix AAA is a transform function, x⃗∈Rn\vec{x}\in\mathbb{R}^nx∈Rn is the input vector, and v⃗\vec{v}v is the output vector. Note however, when rank(A)<n\mathrm{rank}(A)<nrank(A)<n, the output vector is going to be "squished" into a lesser dimensional vector space, i.e. the domain of the transform function is Rn\mathbb{R}^nRn and the range is Rkwherek<n\mathbb{R}^k where k<nRkwherek<n.

Nonsquare matrices[ref] are also spatial transform; it transforms a vector to another vector of a different dimension. This idea is explored in linear mapping section.

Matrix Inverse

Recall Ax⃗=v⃗A\vec{x}=\vec{v}Ax=v, where the square matrix AAA is a spatial transform function that transforms vector x⃗\vec{x}x to v⃗\vec{v}v. If AAA has full rank, each output v⃗\vec{v}v corresponds to a unique input x⃗\vec{x}x, i.e. no dimension "squishing" happens. This means that the matrix mapping fAf_AfA​ is invertible[ref]. x⃗=A−1v⃗\vec{x}=A^{-1}\vec{v} x=A−1v

  • Inverse of n×nn\times nn×n matrix AAA is denoted A−1A^{-1}A−1

Gauss-Jordan Elimination, finding inverse of matrix AAA:

  1. Row reduce the multi-augmented matrix [ A ∣ I ][\ A\ |\ I\ ][ A ∣ I ] so that the left block is in reduced row echelon form.
  2. If the reduced row echelon form is [ I ∣ B ][\ I\ |\ B\ ][ I ∣ B ], then A−1=BA^{-1} = BA−1=B.
  3. If the reduced row echelon form of AAA is not III, then AAA is not invertible.

[ Ax⃗1Ax⃗2Ax⃗3 ]=[ e⃗1e⃗2e⃗3 ][\ A\vec{x}_1\quad A\vec{x}_2\quad A\vec{x}_3\ ]=[\ \vec{e}_1\quad\vec{e}_2\quad\vec{e}_3\ ] [ Ax1​Ax2​Ax3​ ]=[ e1​e2​e3​ ]

We are essentially solving 3 systems of equations simultaneously to get x⃗1,x⃗2,x⃗3\vec{x}_1, \vec{x}_2, \vec{x}_3x1​,x2​,x3​, very elegant.

Change of Basis

Normally, we use vectors associated with standard basis e^1,…,e^n∈Rn\hat{e}_1,\dots, \hat{e}_n\in\mathbb{R}^ne^1​,…,e^n​∈Rn. What if we want to use vectors associated with alternative basis b⃗1,…,b⃗n\vec{b}_1,\dots,\vec{b}_nb1​,…,bn​?

Firstly, given x⃗\vec{x}x in the standard basis, we aim to find the equivalent vector x⃗′\vec{x}'x′ associated with the alternative basis, i.e.

[ b⃗1⋯b⃗n ]x⃗′=[ e^1⋯e^n ]x⃗[\ \vec{b}_1\quad\cdots\quad\vec{b}_n\ ]\vec{x}'=[\ \hat{e}_1\quad\cdots\quad\hat{e}_n\ ]\vec{x} [ b1​⋯bn​ ]x′=[ e^1​⋯e^n​ ]x

The Change of Basis Matrix A=[ b⃗1⋯b⃗n ]A=[\ \vec{b}_1\quad\cdots\quad\vec{b}_n\ ]A=[ b1​⋯bn​ ] is a matrix mapping that transforms vector x⃗′\vec{x}'x′ under alternative basis into vector x⃗\vec{x}x under standard basis.

Ax⃗′=x⃗A\vec{x}'=\vec{x} Ax′=x

Notice this intricate relationship[ref]. We know that x⃗′\vec{x}'x′ is the coordinate under alternative basis (i.e. a contorted coordinate system), while x⃗\vec{x}x is the equivalent coordinate under standard basis. In the mean time, this relationship could also be interpreted as: the original vector x⃗′\vec{x}'x′ under standard basis gets geometrically transformed by AAA into a new vector x⃗\vec{x}x. Understand that there is no inherent conflict between these two intepretations.

It follows that the Inverse Change of Basis Matrix A−1A^{-1}A−1 converts coordinate x⃗\vec{x}x under standard basis into coordinate x⃗′\vec{x}'x′ under alternative basis:

A−1x⃗=x⃗′A^{-1}\vec{x}=\vec{x}' A−1x=x′

Secondly, if we apply a geometric transformation MMM to vector x⃗\vec{x}x under standard basis, how would such transformation manifest on vector x⃗′\vec{x}'x′ under alternative basis?

  1. Start with a coordinate under alternative basis: x⃗′\vec{x}'x′.
  2. We apply a change-of-basis matrix mapping AAA to convert it to a coordinate under standard basis: Ax⃗′A\vec{x}'Ax′
  3. We apply matrix mapping MMM to the vector under standard basis: M(Ax⃗′)M(A\vec{x}')M(Ax′)
  4. We apply an inverse-change-of-basis matrix mapping A−1A^{-1}A−1 to convert it to a coordinate under alternative basis:

    (A−1MA)x⃗′(A^{-1}MA)\vec{x}' (A−1MA)x′

Take special note to the matrix:

A−1MAA^{-1}MA A−1MA

If we operate purely under alternative basis, this is the matrix mapping that performs the same geometric transform as MMM does under standard basis. If A,BA, BA,B are n×nn\times nn×n matrices such that P−1AP=BP^{-1}AP=BP−1AP=B for some invertible matrix PPP, then A,BA, BA,B are said to be similar and have

  • same determinant*
  • same eigenvalues*
  • same rank*
  • same trace*

Determinants

  • Determinant is the volume scaling factor of the linear transformation described by the matrix[ref]. It is also the signed volume of the n-dimensional parallelepiped spanned by the column or row vectors of the matrix[ref]. The sign of the determinant depends on whether the linear mapping preserves the orientation of n-space (e.g. right hand rule).
  • Cofactor of n×nn\times nn×n matrix AAA is defined as

    Cij=(−1)(i+j)det(A(i,j))C_{ij}=(-1)^{(i+j)}\mathrm{det}(A(i,j)) Cij​=(−1)(i+j)det(A(i,j))

    where A(i,j)A(i,j)A(i,j) denotes the submatrix obtained from AAA by deleting the iiith row and jjjth column. Geometrically, ∣Cij∣|C_{ij}|∣Cij​∣ is the volume of an (n−1)(n-1)(n−1) dimensional parallelepiped. This parallelepiped is first spanned by all but jjjth column vector, and then projected to a hyperplane perpendicular to the iiith standard basis e^i\hat{e}_ie^i​.
  • Cofactor Expansion along the first row for an n×nn\times nn×n matrix AAA:

    det(A)=∑k=1na1kC1k\mathrm{det}(A)=\sum_{k=1}^n a_{1k} C_{1k} det(A)=k=1∑n​a1k​C1k​

    Note that cofactor expansion can occur on any row or column.
  • Cofactor Matrix of n×nn\times nn×n matrix AAA is denoted cof(A)\mathrm{cof}(A)cof(A), and

    (cof(A))ij=Cij(\mathrm{cof}(A))_{ij}=C_{ij} (cof(A))ij​=Cij​

Important properties:

  • For an n×nn\times nn×n matrix AAA: det(A)=0 ⟷ rank(A)<n\mathrm{det}(A)=0 \ \longleftrightarrow \ \mathrm{rank}(A)<ndet(A)=0 ⟷ rank(A)<n

  • det(A−1)=[det(A)]−1\mathrm{det}(A^{-1})=[\mathrm{det}(A)]^{-1}det(A−1)=[det(A)]−1
  • det(EC)=det(E) det(C)\mathrm{det}(EC)=\mathrm{det}(E)\ \mathrm{det}(C)det(EC)=det(E) det(C) (makes geometric sense)

Elementary row operations and effects on determinant:

  • If matrix BBB is obtained from AAA by multiplying the iiith row of AAA by r∈Rr\in\Rr∈R, then: det(B)=r det(A)\mathrm{det}(B)=r\ \mathrm{det}(A)det(B)=r det(A) (makes geometric sense)
  • If matrix BBB is obtained from AAA by swapping two arbitrary rows, then det(B)=−det(A)\mathrm{det}(B)=-\mathrm{det}(A)det(B)=−det(A) (by induction, also makes geometric sense with right hand rule)
  • If matrix BBB is obtained from AAA by adding rrr times iiith row of AAA to kkkth row, then det(B)=det(A)\mathrm{det}(B)=\mathrm{det}(A)det(B)=det(A) (by induction, also makes geometric sense)

From these properties, we derive an effective strategy for evaluating the determinant of a matrix: simply row reduce the matrix to upper-triangular form and then evaluate diagonal product.

Eigenvectors

Given linear mapping L:Rn→RnL:\R^n\rightarrow\R^nL:Rn→Rn, we have non-zero vector v⃗\vec{v}v and scalar λ\lambdaλ that satisfies

L(v⃗)=λv⃗L(\vec{v})=\lambda \vec{v} L(v)=λv

Or if we use square matrix AAA:

Av⃗=λv⃗A\vec{v}=\lambda \vec{v} Av=λv

  • Engenvector: v⃗\vec{v}v; geometrically, it is a vector whose direction is preserved after a linear transform.
  • Eigenvalue: λ\lambdaλ
  • Eigenpair: tuple (λ,v⃗)(\lambda, \vec{v})(λ,v)
  • Eigenspace of λ\lambdaλ: the set containing zero vector and all eigenvectors associated with λ\lambdaλ. Note that an eigenspace could be two 1D lines intersecting at origin.
  • Eigenbasis: the set of eigenvectors of n×nn\times nn×n matrix AAA that forms a basis of Rn\R^nRn.

Finding Eigenvectors:

Av⃗−λv⃗=0⃗(A−λI)v⃗=0⃗det(A−λI)=0[ solve for λ ]\begin{aligned} A\vec{v} - \lambda \vec{v}&=\vec{0}\\ (A-\lambda I)\vec{v}&=\vec{0}\\ \mathrm{det}(A-\lambda I) &= 0\qquad[\ \textrm{solve for }\lambda\ ] \end{aligned} Av−λv(A−λI)vdet(A−λI)​=0=0=0[ solve for λ ]​

  • Characteristic Polynomial of matrix AAA:

    C(λ)=det(A−λI)C(\lambda)=\mathrm{det}(A-\lambda I) C(λ)=det(A−λI)

    The roots of C(λ)C(\lambda)C(λ) are the eigenvalues of AAA.
  • Algebraic Multiplicity of λ\lambdaλ: the number of times λ\lambdaλ is repeated as a root of the characteristic polynomial.
  • Geometric Multiplicity of λ\lambdaλ: the dimension of the eigenspace of λ\lambdaλ
  • Deficient eigenvalue: whose geometric multiplicity < algebraic multiplicity, e.g. 2d proj\mathrm{proj}proj transform, 2d shear transform both have 2 algebraic multiplicity but 1 geometric multiplicity.

Properties of characteristic polynomials:

  • Total number of roots is nnn, counting complex and repetitions.
  • Complex roots occur in "conjugate pairs"
  • If nnn is odd, there must be at least one real root*
  • The constant term in the characteristic polynomial is det(A)\mathrm{det}(A)det(A)

Properties of eigenvalue:

  • For eigenvalue λ\lambdaλ: 1≤geometric multiplicity≤algebraic multiplicity1\le \textrm{geometric multiplicity}\le \textrm{algebraic multiplicity}1≤geometric multiplicity≤algebraic multiplicity*
  • Eigenvectors from eigenspaces of different eigenvalues are necessarily linearly independent (by induction*).
  • If kkk distinct eigen values λ1,…,λk\lambda_1,\dots,\lambda_kλ1​,…,λk​ of n×nn\times nn×n matrix AAA all have property geometric multiplicity = algebraic multiplicity, then the collection of basis from all eigenspaces forms eigenbasis in Rn\R^nRn. (see Diagonalization Theorem)

Diagonalization

Suppose n×nn\times nn×n matrix AAA has eigenbasis {v⃗1,…,v⃗n}\{\vec{v}_1,\dots,\vec{v}_n\}{v1​,…,vn​} corresponding to eigenvalues λ1,…,λn\lambda_1, \dots, \lambda_nλ1​,…,λn​, if we let P=[ v⃗1 ⋯ v⃗n ]P=[\ \vec{v}_1\ \cdots\ \vec{v}_n\ ]P=[ v1​ ⋯ vn​ ], we have

AP=A[ v⃗1⋯v⃗n ]=[ Av⃗1⋯Av⃗n ]=[ λ1v⃗1⋯λnv⃗n ]=P diag(λ1,…,λn)\begin{aligned} AP&= A[\ \vec{v}_1\quad \cdots\quad \vec{v}_n\ ]\\ &=[\ A\vec{v}_1\quad \cdots\quad A\vec{v}_n\ ]\\ &=[\ \lambda_1\vec{v}_1\quad \cdots\quad \lambda_n\vec{v}_n\ ]\\ &=P\ \mathrm{diag}(\lambda_1, \dots, \lambda_n) \end{aligned} AP​=A[ v1​⋯vn​ ]=[ Av1​⋯Avn​ ]=[ λ1​v1​⋯λn​vn​ ]=P diag(λ1​,…,λn​)​

If there exists an invertible matrix PPP and a diagonal matrix DDD such that

P−1AP=D=diag(λ1,…,λn)P^{-1} AP=D=\mathrm{diag}(\lambda_1,\dots, \lambda_n) P−1AP=D=diag(λ1​,…,λn​)

we say that AAA is diagonalizable, and that matrix PPP diagonalizes AAA to its diagonal form DDD. Geometrically, P−1APP^{-1} APP−1AP is performing the same geometric transform as AAA under eigenbasis.

Diagonalization Theorem: n×nn\times nn×n matrix AAA can be diagonalized if and only if there exists an eigenbasis {v⃗1,…,v⃗n}\{\vec{v}_1, \dots, \vec{v}_n\}{v1​,…,vn​} for Rn\R^nRn. Then, the matrix P=[ v⃗1 ⋯ v⃗n ]P=[\ \vec{v}_1\ \cdots\ \vec{v}_n\ ]P=[ v1​ ⋯ vn​ ] diagonalizes AAA to a diagonal matrix D=diag(λ1,…,λn)D=\mathrm{diag}(\lambda_1,\dots, \lambda_n)D=diag(λ1​,…,λn​), where λi\lambda_iλi​ corresponds to v⃗i\vec{v}_ivi​.

Relevant properties:

  • A matrix AAA is diagonalizable if and only if every eigen value of matrix AAA has its geometric multiplicity equal to its algebrac multiplicity.
  • If an n×nn\times nn×n matrix AAA has distinct eigenvalues, then AAA is diagonalizable.
