Preface

This is a comprehensive overview of multivariable calculus and differential equations that includes detailed proof and formulations. Readers' knowledge in Set Theory, Linear Algebra and Single Variable Calculus is assumed. The content of this overview is primarily based on MIT OpenCourseWare 18.02 Calculus of Several Variables.

Exercises and examples are normally not included in this purely conceptual overview. Nevertheless, they may be included occassionally to get certain points across.

Open Set and Closed Set

Set A∈RnA \in \mathbb{R}^nA∈Rn is an Open Set if

In other words, set A is open if it contains an open ball about each of its points.

A Closed Set is the complement of an Open Set. Set B∈RnB \in \mathbb{R}^nB∈Rn is closed if and only if B contains all of its limit points.

Examples:

  • Closed interval [0,1][0,1][0,1] is a closed set because it contains all of its limit points.
  • Rn−{0}\mathbb{R}^n-\{0\}Rn−{0} is open because it does not contain its limit point 111.
