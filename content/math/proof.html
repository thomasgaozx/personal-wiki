If AAA and BBB are 2 sets, the cross product of AAA and BBB, written A×BA\times BA×B, is the set of all ordered pairs wherein the first element ∈A\in A∈A and the second element ∈B\in B∈B.

Equivalence Relation

A binary relation RRR is an equivalence relation if RRR satisfies three conditions:

  1. R is reflexive if xRx∀xxRx\forall xxRx∀x
  2. R is symmetric if xRy⇒yRx∀xxRy\Rightarrow yRx \forall xxRy⇒yRx∀x
  3. R is transistive if xRy∧yRz⇒xRz∀x,y,zxRy\wedge yRz\Rightarrow xRz\forall x,y,zxRy∧yRz⇒xRz∀x,y,z

Proof by Contradiction

Useful to prove a rule is true for all objects (use common logic to find why the statement is true/false, then use proof by contradiction as media to demonstrate).

Prove that 2\sqrt{2}2​ is irrational. (A rational number can be reduced to fraction m/nm/nm/n where m,n∈Zm, n\in \mathbb{Z}m,n∈Z)

2=m/n,m≢n(mod 2)⇒2n2=m2\sqrt{2}=m/n, m\not\equiv n (\text{mod 2})\Rightarrow 2n^2=m^2 2​=m/n,m​≡n(mod 2)⇒2n2=m2

Clearly mmm is even, let m=2km=2km=2k, we have

2n2=(2k)2⇒n2=2k22n^2=(2k)^2\Rightarrow n^2=2k^2 2n2=(2k)2⇒n2=2k2

Clearly nnn is even. This violates the rule m≢n(mod 2)m\not\equiv n (\textrm{mod 2})m​≡n(mod 2) and we have a contradiction.

Show that every graph with two or more nodes contains two nodes that have equal degrees:

Consider graph G=(V,E)G=(V,E)G=(V,E), let n=count(V)≥2n=\textrm{count}(V)\ge 2n=count(V)≥2, the minimum degree of a node is 0 and the maximum degree of a node is n−1n-1n−1. Assume that all nodes have unique degree, then the degree of each node has to be exactly from 0 to n−1n-1n−1. However, if a node has degree n−1n-1n−1, then all nodes are linked and there cannot be a node that has degree 0. This yields a conradiction.

As you can see, proof of contradiction really is just a "media". The difficult part of the proof is still figuring out the logic. Often the logic is linked to the property of the object given in the statement. It will most likely help if one carefully examine the basic properties.

Induction

  • Base Case: prove the statement holds for some finite set of objects.
  • Induction Step: assume statement holds for an object, prove that statement holds for the subsequent object(s)
  • Inductive Hypothesis: the assumption that statement holds in inductive step.
