Preliminaries Definitions

A graph with eee edges and nnn nodes is denoted G(n,e)G(n,e)G(n,e). An oriented linear graph is a collection of directed edges (listed numerically, we use 1,2,…1,2,\dots1,2,… to label edges) that connect at nodes (listed alphabetically, we use a,b,c,…a,b,c,\dotsa,b,c,…)

  • Self Loop: an edge from a node to itself
  • Subgraph: graph GGG is a subgraph of HHH if the nodes of GGG are a subset of the nodes of HHH, and the edge of GGG are the edges of HHH on the corresponding nodes.
  • Complement of subgraph: original graph - subgraph.
  • Path: a sequence of nodes connected by edges.
  • Simple Path: a path that doesn't repeat any nodes
  • Cycle: a path that starts and ends in the same node
  • Connected Graph: a graph is connected if every two nodes have a path between them.
  • Isomorphism: graph that are topologically equivalent. (same all-row incidence matrix) Both directed and undirected graph GGG are often represented by (V,E)(V,E)(V,E) where VVV is a set of vertices and EEE is a set of edges.

F-Cutsets and F-Circuits

  • Tree: contains all nodes of the graph, is connected, and contains no circuits (no closed loop). There are many possible trees for a given graph.
    • Branch: edge of a tree; given graph G(n,e)G(n,e)G(n,e) of ppp parts

    #Branches=n−p\textrm{\#Branches} = n-p #Branches=n−p

  • Co-tree: the complement to a tree
    • Chord: an edge in a co-tree
  • Forests: a collection of trees for graphs with more than one part.

Cutset vs Circuit:

  • Cutset: a set of edges that upon removal, the connected graph split into two parts. Moreover, none of the subset of the cut set has this property.
    • Nodal cutsets: segregate a single node from the remaining node.
    • Fundamental cutsets (f-cutsets): cutsets that only contains one branch, and the remaining edges are chords; the positive direction of the cutset is set by the branch.
  • Circuit: a set of edges that form exactly 1 loop; every node in the circuit is incident on exactly two edges.
    • Fundamental circuits (f-circuits): circuits that contain exactly one chord and some branches; the direction of the f-circuit is set by the chord.

Theorem 1: for any trees in a connected graph G(n,e)G(n,e)G(n,e) with incidence matrix AAA, the columns representing the branches in AAA have rank n−1n-1n−1.

KCL Theorem: the orientated sum of the through measurements implied by the edges in a cutset is zero at any instant in time

∑flow in=∑flow out\sum \textrm{flow in}=\sum\textrm{flow out} ∑flow in=∑flow out

System Models

System Variables

These are the time-dependent qquantities that describes the state of the system. Examples are displacement, voltage, current, force, etc. Later we will formalize this when we talk about state-space. All of the system variables we will consider can be classified as either:

  • Across variables: represents a property that can be measured across a component; voltage, temperature, pressure and velocity, displacement, acceleration. Across variables are stored in across components.
  • Through variables: represents a property that appears to flow through a component; measured at a point; current, heat flow, fluid flow, and force. Through variables are stored in through components.

System components are the "building blocks" of a system, characterized by the processes involved, the terminals of interconnection and the types of variables that can be measured. They are an idealization of a real process (main assumptions: linear, time-invariant, lumped, deterministic).

Electrical Circuitry

The analysis procedure to find expression for V/I of a particular component:

  1. Draw the linear graph
  2. Component equation:
    • sources (known)
    • equation for each component (e.g. resistor, capacitor, etc.)
  3. Pick the tree such that sources and interested component edge are in the tree
  4. Write the f-cutset and f-circuit equations
    • f-cutset: KCL
    • f-circuit: KVL
  5. Write out the matrix form of KCL/KVL.
  6. Formal: use branch transformation or chord transformation. Casual: just by observation.
  7. Substitute component equation to obtain the final differential equation (containing only 1 unknown).

Electrical circuits to graph conventions:

  • Sources are chords, same direction as source direction

Diagonalization Technique
