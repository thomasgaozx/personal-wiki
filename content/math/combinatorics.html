Bijection and Inverse

A bijection is a function f:S→Tf:S\rightarrow Tf:S→T such that

  1. fff is "1-1" or injective: f(x1)=f(x2)⇒x1=x2f(x_1)=f(x_2)\Rightarrow x_1=x_2f(x1​)=f(x2​)⇒x1​=x2​
  2. fff is "onto" or surjective: ∀ y∈T ∃ x∈S\forall\ y\in T\ \exists\ x\in S∀ y∈T ∃ x∈S such that f(x)=yf(x)=yf(x)=y

Theorem 1: Suppose S,TS,TS,T are finite sets and f:S→Tf:S\rightarrow Tf:S→T:

  1. If fff is "1-1", then ∣S∣≤∣T∣|S|\le |T|∣S∣≤∣T∣ (each element of SSS is mapped to a distinct element of TTT)
  2. If fff is "onto", then ∣S∣≥∣T∣|S|\ge |T|∣S∣≥∣T∣ (each element of TTT is the image of some element in SSS)
  3. If fff is a bijection, then ∣S∣=∣T∣|S|=|T|∣S∣=∣T∣

Definition: An inverse of f:S→Tf:S\rightarrow Tf:S→T is a function f−1:T→Sf^{-1}:T\rightarrow Sf−1:T→S such that

  1. ∀ x∈S,f−1(f(x))=x\forall\ x\in S, f^{-1}(f(x)) = x∀ x∈S,f−1(f(x))=x
  2. ∀ y∈T,f(f−1(y))=y\forall\ y\in T, f(f^{-1}(y)) = y∀ y∈T,f(f−1(y))=y

Theorem 2: A function f:s→Tf:s\rightarrow Tf:s→T has an inverse if and only if it is a bijection. Proof:

  1. From "1-1": suppose f(x1)=f(x2)f(x_1)=f(x_2)f(x1​)=f(x2​), we know f−1(f(x1))=f−1(f(x2))f^{-1}(f(x_1))=f^{-1}(f(x_2))f−1(f(x1​))=f−1(f(x2​)), (x1=x2x_1=x_2x1​=x2​)
