Laplace Transform

There are 3 primary motivations for using Laplace transform:

  • Determining sinusoidal response from sinusoidal input
  • Knowing x(t)x\left( t \right)x(t) and h(t)h\left( t \right)h(t), determining y(t)y\left( t \right)y(t)
  • Knowing x(t)x\left( t \right)x(t) and y(t)y\left( t \right)y(t), determining h(t)h\left( t \right)h(t)

Determining Output

Knowing x(t)x\left( t \right)x(t) and h(t)h\left( t \right)h(t), the output

y(t)=x(t)∗h(t)y\left( t \right) = x\left( t \right)*h\left( t \right) y(t)=x(t)∗h(t)

Y(s)=X(s)∗h(s)Y\left( s \right) = X\left( s \right)*h\left( s \right) Y(s)=X(s)∗h(s)

Determining Impulse Response

Given a complicated differential equation,

  1. We Laplace transform both sides
  2. Rearrange to obtain an expression for Y(s)/X(s)Y\left( s \right)/X\left( s \right)Y(s)/X(s)
  3. Inverse Laplace transform the obtained expression, and we have h(t)h\left( t \right)h(t)

Region of Convergence

The ROC is the region of complex plane (sss or zzz-plane) for which the L\mathcal{L}L or Z\mathcal{Z}Z converges. We don’t have to manually derive an expression to find the region of convergence!

Five simple rules to find ROC (Source):

  1. ROC must be bounded by poles or extends to infinity.
  2. If the signal in time-domain is right-sided, ROC is the right side of rightmost pole, (or for Z transform, outside the rim bounded by the outermost pole).
  3. If the signal in time-domain is left-sided, ROC is the left side of the leftmost pole, (or for Z transform, inside the rim of the innermost pole).
  4. If the signal in time-domain is two-sided, ROC is bounded by poles.
  5. If the signal in time-domain has a finite duration, ROC is everywhere.

Comparison of Stability

Asymptotic Stability aka Internal Stability

For Laplace Transform:

  • Internally stable only if and only if all poles are in the left-half plane.
  • Marginally stable if there are unrepeated poles on the imaginary axis.
  • Unstable otherwise.

For Z Transform:

  • Internally stable only if and only if all poles are within the unit circle.
  • Marginally stable if there are unrepeated poles on the unit circle.
  • Unstable otherwise.

For both CT and DT Systems:

  • If a system is marginally stable or internally unstable, the system is BIBO unstable.
  • An asymptotically (internally) stable system is BIBO stable.
