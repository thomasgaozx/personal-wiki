Classification of Signals

C.T. vs D.T.

Continuous-Time (C.T.) Signal is defined over a continuous range of time. It is denoted as x(t)x(t)x(t) vs ttt.

Discrete-Time (D.T.) Signal is a sequence of numbers. The signal is specified at discrete values of time. It is denoted as x[n]x[n]x[n] vs nnn.

System Classifications

C.T vs D.T., Analog vs Digital

C.T. Systems have C.T. input and output signals.

D.T. Systems have D.T. input and output signals.

Analog systems have analog input and output signals.

Digital systems have digital input and output signals.

Time-Invariant vs Time-Varying Systems

Time-invariant or constant parameter system: its parameters don't change with time. If the input is delayed by TTT, the output is the same but delayed by TTT.

{x(t)⇒systemy(t)⇔x(t−T)⇒systemy(t−T)x[n]⇒systemy[n]⇔x[n−m]⇒systemy[n−m]\begin{cases} x(t)\xRightarrow{\textrm{system}}y(t)\Leftrightarrow x(t-T)\xRightarrow{\textrm{system}}y(t-T)\\ x[n]\xRightarrow{\textrm{system}}y[n]\Leftrightarrow x[n-m]\xRightarrow{\textrm{system}}y[n-m] \end{cases} {x(t)system​y(t)⇔x(t−T)system​y(t−T)x[n]system​y[n]⇔x[n−m]system​y[n−m]​

Time-varying systems are not time-invariant.

Procedure to test time variance:

  1. Delay input, change all ttt inside x(… )x(\dots)x(…) to t+δt+\deltat+δ, and obtain y1(t)y_1(t)y1​(t).
  2. Delay output, change all ttt to t+δt+\deltat+δ, and obtain y2(t)y_2(t)y2​(t)
  3. Compare, y1=y2⇒y_1=y_2\Rightarrowy1​=y2​⇒ Time Invariant

Memoryless vs Dynamic (with memory) Systems

Memoryless Systems: the output at ttt only depends on the input at time ttt.

System with Memory: the output depends on past inputs.

System with finite memory of TTT seconds: output are determined by the inputs in last TTT seconds.

System with infinite memory: output at time ttt depends on inputs over the past (−∞,t](-\infty,t](−∞,t]

  • y(t)=(t−1)x(t)y(t)=(t-1)x(t)y(t)=(t−1)x(t) is memoryless
  • y(t)=(t−1)x(t)+x(t−3)y(t)=(t-1)x(t) + x(t-3)y(t)=(t−1)x(t)+x(t−3) has finite memory of 333s
  • y[n]=y[n−1]+x[n]=∑k=−∞nx[k]\displaystyle y[n]=y[n-1]+x[n]=\sum_{k=-\infty}^n x[k]y[n]=y[n−1]+x[n]=k=−∞∑n​x[k] (accumulator function) has infinite memory

Causal vs Noncausal Systems

Causal systems: output only depends on past and present input (outputs cannot start before the inputs start). E.g. y(t)=x(t)+x(t−2)y(t)=x(t)+x(t-2)y(t)=x(t)+x(t−2) and y(t)=x(t)sin⁡(t+2)y(t)=x(t)\sin(t+2)y(t)=x(t)sin(t+2) are causal because they only depends on input at time ttt.

Noncausal systems: output starts before the input starts. e.g. y(t)=x(t+1)−x(t)y(t)=x(t+1)-x(t)y(t)=x(t+1)−x(t) and y[n]=x[n+3]+x[n−3]y[n]=x[n+3]+x[n-3]y[n]=x[n+3]+x[n−3] are noncausal.

Ignore the examples, just check the graph of output y(t)y(t)y(t) and input x(t)x(t)x(t) and see who started first.

Noncausal systems are not realizable in real time but can be useful for recorded data, e.g. a moving average system.

Invertible vs Noninvertible Systems

A system is invertible if its input can be obtained (recovered) from its output. (There should be one-to-one mapping between input and output.)

x(t)⇒Sy(t)⇒Six(t)x(t)\xRightarrow{S}y(t)\xRightarrow{S_i}x(t) x(t)S​y(t)Si​​x(t)

Non-invertible systems do not have one-to-one mapping between input and output, e.g. y(t)=x2(t)y(t)=x^2(t)y(t)=x2(t) gets rid of the negative sign and is non-invertible.

Stable vs Unstable Systems

A system is Bounded Input/Bounded Output (BIBO) stable if and only if every bounded input results in bounded output.

{C.T.:∣x(t)∣≤Mx<∞⇒∣y(t)∣≤My<∞D.T.:∣x[n]∣≤Mx<∞⇒∣y[t]∣≤My<∞\begin{cases} \textrm{C.T.:}&|x(t)|\le M_x <\infty &\Rightarrow\quad |y(t)|\le M_y< \infty\\ \textrm{D.T.:}&|x[n]|\le M_x <\infty &\Rightarrow\quad |y[t]|\le M_y< \infty \end{cases} {C.T.:D.T.:​∣x(t)∣≤Mx​<∞∣x[n]∣≤Mx​<∞​⇒∣y(t)∣≤My​<∞⇒∣y[t]∣≤My​<∞​

Otherwise, the system is unstable, e.g. ∣y(t)∣→∞|y(t)|\rightarrow\infty∣y(t)∣→∞ as t→∞t\rightarrow\inftyt→∞ even when x(t)x(t)x(t) is bounded.

Note that time should be the independent variable, not the input.

Linear vs Nonlinear Systems

Linearity conditions:

  • Additive Property: {x1→y1x2→y2⟹x1+x2→y1+y2\begin{cases}x_1\rightarrow y_1\\x_2\rightarrow y_2\end{cases}\Longrightarrow x_1+x_2\rightarrow y_1+y_2{x1​→y1​x2​→y2​​⟹x1​+x2​→y1​+y2​
  • Scaling Property: x→y⇒kx→kyx\rightarrow y\Rightarrow kx\rightarrow kyx→y⇒kx→ky
  • Superposition combines the additive and scaling properties.

Nonlinear Systems do not comply with one or boths of the properties.

Importance of impulse response of linear systems: Let an arbitrary input x(t)=a1x1(t)+a2x2(t)+⋯+amxm(t)x(t)=a_1x_1(t)+a_2x_2(t)+\cdots+a_mx_m(t)x(t)=a1​x1​(t)+a2​x2​(t)+⋯+am​xm​(t). For a linear system the response should be y(t)=a1y1(t)+a2y2(t)+⋯+amym(t)y(t)=a_1y_1(t)+a_2y_2(t)+\cdots +a_my_m(t)y(t)=a1​y1​(t)+a2​y2​(t)+⋯+am​ym​(t).

We can approximate x(t)x(t)x(t) as a sum of rectangular pulses. The approximation improves as Δt→0\Delta t\rightarrow 0Δt→0 (impulse). So if we know the impulse response of a linear time invariant (LTI) system, we know its response to any input.
